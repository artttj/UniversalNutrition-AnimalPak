# SomethingDigital_SidebarMinicart

## Purpose

This module enables a full-height slide-in minicart. The minicart utilizes Magento's modal logic (with slide-in setting) instead of the default dropdown popover that comes OOB. This setting is useful for stores where purchasing multiple items is quite common.

## Usage

To enable this module, go to the Magento admin then navigate to `Stores -> Configuration -> Sales -> Checkout Shopping Cart Sidebar` and select **Yes** for **Enable Minicart Slide-in Minicart**.
